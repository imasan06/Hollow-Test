## ¿Qué es Hollow?

Hollow es la aplicación compañera del smartwatch **Hollow 0W**. Desarrollada con React y empaquetada como app nativa con Capacitor, permite conectarse al reloj mediante **Bluetooth Low Energy (BLE)** para visualizar datos, configurar el dispositivo y mantenerse sincronizado en tiempo real.

La app corre tanto en **Android** como en **web**, con una sola base de código.

---

## ¿Qué puedes hacer con Hollow?

📡 **Conectar tu reloj** — Escanea y empareja tu Hollow watch vía Bluetooth LE de forma automática.

📊 **Visualizar tus datos** — Accede a la información que registra tu reloj en tiempo real con gráficas claras.

⚙️ **Configurar tu dispositivo** — Ajusta las preferencias del reloj directamente desde la app.

💾 **Guardar preferencias** — Tu configuración y datos se almacenan de forma persistente en el dispositivo.

## Stack tecnológico

Hollow está construida con un stack moderno de frontend que se empaqueta como app móvil nativa:

- **UI** → React 18 + TypeScript + Vite
- **Estilos** → Tailwind CSS + shadcn/ui + Radix UI
- **Empaquetado móvil** → Capacitor (Android / iOS)
- **Bluetooth** → @capacitor-community/bluetooth-le
- **Gestión de estado** → TanStack Query + React Hook Form + Zod
- **Routing** → React Router DOM
- **Gráficas** → Recharts
- **Almacenamiento** → @capacitor/preferences

## Contexto del proyecto

Hollow nació como el software compañero de un smartwatch propio. La idea es simple: el hardware solo tiene sentido si la app que lo acompaña está a la altura. Por eso Hollow está pensada para ser rápida, confiable y fácil de usar, con una interfaz limpia que pone los datos del reloj en primer plano.

El proyecto está en etapa de desarrollo activo, con 71+ commits y enfoque en la conexión BLE y la experiencia de usuario móvil.
